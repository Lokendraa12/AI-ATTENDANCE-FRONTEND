# AI Face Recognition Attendance System

A smart web-based attendance management system that uses AI face recognition technology to automatically mark student attendance.

## 🚀 Features

- AI Face Recognition Attendance
- Real-time Face Detection
- Admin Dashboard
- Student Management
- Attendance Reports
- Attendance Percentage
- Search & Filter Students
- Modern Responsive UI
- Private & Demo Login
- MongoDB Database Integration
- Face Image Upload
- Attendance History

# 🧠 How It Works

1. Admin uploads student face image.
2. Camera scans live student face.
3. face-api.js compares face descriptors.
4. If matched, attendance is marked.
5. Attendance stored in MongoDB.
